# How to run the webpage (AI chatbot)
How to open the webpage:
1. Download the project files from "main" branch.
2. Open the folder on your IDE
3. Navigate to homepageUI folder, open homepage.html and run locally (for VSCode, use LiveServer)
4. The homepage should show up with login/logout and signup button at the top right corner and ready for account creation, login/lougout testing,...
5. On the bottom, there is a popup chatbot button. Click on the button and a chat window will appear.
